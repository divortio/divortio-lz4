LZ4 Web Examples

Select a demo below to test the LZ4 JS implementation.

1. Browser File Processing

Select a local file, compress it in the browser using LZ4, and download the result. No server upload involved.

2. Stream Upload to Server

Compress a file in the browser as a stream and upload it directly to this Node.js server.

File: lz4.stream.upload-server.html

3. OPFS Stream (Origin Private File System)

Stream compression directly to the browser's persistent private storage (OPFS). Useful for large local datasets.

File: lz4.stream.opfs.html

4. Fetch Stream Download

Download a compressed stream from the server and decompress it on the fly.

File: lz4.stream.fetch-download.html

5. Web Worker (Off-Main-Thread)

Uses a background thread to compress/decompress large data without freezing the UI. Supports SharedArrayBuffer for zero-copy performance.
